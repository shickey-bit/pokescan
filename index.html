Cloudflare Worker proxy — required
Browsers block direct AI API calls for security. A free Cloudflare Worker acts as a secure middleman. Takes about 3 minutes. Follow the steps below.
Setup guide — 5 steps
1
Go to dash.cloudflare.com/sign-up and create a free account. No credit card needed.
2
Workers & Pages → Create → Create Worker → name it pokescan-proxy → Deploy
3
Click Edit code, select all, delete it, paste the worker code below, then click Deploy
4
Worker Settings → Variables & Secrets → Add → name: ANTHROPIC_API_KEY → paste your key from console.anthropic.com → click Encrypt → Save
5
Copy your Worker URL (looks like pokescan-proxy.xyz.workers.dev) and paste it in the field above
